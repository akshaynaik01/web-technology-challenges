# JavaScript Challenges

This folder contains JavaScript coding challenges for practicing fundamental web development skills.

## Challenges Overview

### 1. String Manipulation (Easy)
**File:** `01-string-manipulation.js`

Practice basic string methods and manipulation techniques:
- Reverse a string
- Count character occurrences
- Remove duplicate characters

**Skills:** String methods (split, join, reverse), loops, conditionals

---

### 2. Array Methods (Medium)
**File:** `02-array-methods.js`

Master array manipulation with higher-order functions:
- Filter even numbers
- Double array values
- Sum array elements
- Find maximum value

**Skills:** Array methods (map, filter, reduce), iteration, logic

---

### 3. DOM Manipulation (Intermediate)
**File:** `03-dom-manipulation.js`

Learn to interact with the Document Object Model:
- Select and modify elements
- Create and append new elements
- Add event listeners
- Remove elements from DOM
- Toggle CSS classes

**Skills:** DOM selection (getElementById, querySelector), event handling, class manipulation

---

## How to Use

1. Open each challenge file in your browser console or Node.js environment
2. Read the challenge description and requirements
3. Implement the function based on the provided examples
4. Test your code with the provided test cases
5. Run `console.log()` to verify your solutions

## Example Test Run

```javascript
// Load the challenge file
// Test string manipulation
console.log(reverseString('hello')); // Output: 'olleh'
console.log(countCharacter('hello world', 'l')); // Output: 3
console.log(removeDuplicates('aabbcc')); // Output: 'abc'
```

## Difficulty Levels

- **Easy:** Basic functions and string/array operations
- **Medium:** Higher-order functions and array methods
- **Intermediate:** DOM manipulation and event handling

## Tips for Success

- Start with Easy challenges before moving to Medium
- Use browser DevTools to debug
- Practice these concepts regularly
- Try solving without looking at hints
- Create variations of the challenges

## Getting Started

For beginners, start with `01-string-manipulation.js` and progress through the files.
