Challenge 15: Responsive Image Gallery

Task:

Create a responsive image gallery with lightbox functionality and hover effects.

Requirements:

  • Use CSS Grid for responsive layout
  • Add hover effects to gallery items
  • Implement lightbox to view images in full size
  • Add close button for lightbox
  • Make it fully responsive for mobile devices
