Web Components Challenge

Intermediate

1. Create a Custom Card Component

Create a custom HTML element called <custom-card> that displays a reusable card component.

Requirements:

  • Create a Web Component class that extends HTMLElement
  • Use Shadow DOM to encapsulate styles
  • Accept attributes for title, description, and image URL
  • Implement proper styling within the Shadow DOM
  • Add event listeners for card interactions
<custom-card title="Web Development" description="Learn modern web technologies" image="https://example.com/image.jpg"> </custom-card>
Advanced

2. Create a Data Binding Component

Create a Web Component that implements two-way data binding between form inputs and component state.

Requirements:

  • Create a Web Component for form binding
  • Implement reactive property changes
  • Use custom events for data updates
  • Validate input data
  • Support multiple input fields
<data-binding-form id="myForm"> <input name="email" type="email" /> <input name="username" type="text" /> </data-binding-form>
Intermediate

3. Create a Slot-based Layout Component

Create a Web Component that uses slots to create a flexible layout structure.

Requirements:

  • Create a layout Web Component using named slots
  • Support header, main content, and sidebar slots
  • Make the layout responsive
  • Apply consistent styling
  • Allow slot manipulation from parent
<layout-component> <div slot="header">Header Content</div> <div slot="main">Main Content</div> <div slot="sidebar">Sidebar</div> </layout-component>
Advanced

4. Build a Custom Form Component

Create a complete custom form component with validation, submission, and custom validation rules.

Requirements:

  • Create a Web Component for form handling
  • Implement built-in and custom validation
  • Support form submission with data serialization
  • Add error message display
  • Implement reset functionality
  • Support async validation
<custom-form id="registrationForm"> <input type="text" name="username" required /> <input type="email" name="email" required /> <input type="password" name="password" required /> <button type="submit">Register</button> </custom-form>
Advanced

5. Implement Lifecycle Hooks

Create a Web Component that demonstrates proper lifecycle management and state handling.

Requirements:

  • Implement all lifecycle hooks (constructor, connectedCallback, disconnectedCallback, adoptedCallback)
  • Handle property changes with propertyChangedCallback
  • Manage internal state properly
  • Clean up resources in disconnectedCallback
  • Support attribute observation
  • Demonstrate proper memory management
lifecycleComponent.observedAttributes = ['data-count', 'data-name']; componentElement.setAttribute('data-count', '10');
