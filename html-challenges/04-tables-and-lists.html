Student Grades Table

Task: Create a table showing student information with the following columns:

  • Student Name
  • Roll Number
  • Math Score
  • Science Score
  • English Score
  • Total Score

Requirements:

  1. Use proper table headers with <thead> and <th>
  2. Add at least 5 student records in <tbody>
  3. Add a footer row with <tfoot> showing average scores
  4. Use colspan for the "Average:" label in footer
  5. Add border and padding using inline styles
